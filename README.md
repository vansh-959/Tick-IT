Tick-IT (Ticket & Task Management System)

Tick-IT is a production-ready, high-performance ticket and task management system built using modern web technologies. Designed for seamless local client-side performance, it allows IT helpdesks, support teams, and personal users to efficiently create, search, filter, and track support tickets with zero server dependency and zero hosting costs.

🔗 Project Links

Live Web App (Vercel): https://tick-it-alpha.vercel.app/

GitHub Repository: https://github.com/vansh-959/Tick-IT

Mobile App Landing Page (Google Sites): https://sites.google.com/view/tickit-tasks

🚀 Key Features

Offline-First Architecture: Fully functional offline using secure browser-side LocalStorage cache persistence.

Zero Server Hosting Costs: Stores records securely in browser storage, eliminating expensive database servers.

Interactive UI/UX: Premium dark-mode dashboard built with responsive Tailwind CSS utilities.

Dynamic Pipelines: Real-time search processing, priority-level filtering, and instant status updates.

Zero Loading Lag: Fast UI state updates without page refreshes, resulting in instantaneous data syncs.

📱 Mobile App Details (Android Platform)

Through modular React engineering, the codebase has been compiled into a fully native Android APK:

Framework: CapacitorJS Wrapper

APK Size: ~5.6 MB (Ultra-Lightweight & optimized)

Database Sync: Synchronized browser WebView LocalStorage database engine

Package Name: com.vanshdeep.tickit

Installation: Download directly from our Google Sites Landing Page with a single click
